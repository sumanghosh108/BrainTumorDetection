## Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| Weighted ensemble over single model | Reduces misclassification on edge cases; weights tunable via env vars |
| YOLO detection before classification | Localises the tumor region вҶ’ classifiers see less noise |
| Grad-CAM on ResNet101 specifically | Highest-weighted backbone; provides most trustworthy explanation |
| asyncio.to_thread for inference | Keeps the FastAPI event loop responsive during CPU-bound model work |
| Pre-signed S3 URLs | Avoids making the bucket public; URLs expire after 1 hour |
| Structured audit log | Dual-write (DB + stdout) for compliance and log aggregation |
